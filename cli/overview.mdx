PairOfCleats provides a comprehensive CLI for indexing, searching, and managing code repositories.

## Installation

```bash
npm install -g pairofcleats
```

## Quick Start

```bash
# Run guided setup
pairofcleats setup

# Build an index
pairofcleats index build

# Search your code
pairofcleats search "function name"
```

### Backend Commands

**LMDB build** - Build LMDB materialized index for fast queries:

```bash
pairofcleats lmdb build --repo . --mode code
```

<ParamField path="--mode" type="string">
  Mode to build: code, prose, extracted-prose, records
</ParamField>

<ParamField path="--index-root" type="string">
  Override index root directory
</ParamField>

<ParamField path="--snapshot" type="string">
  Build from specific snapshot
</ParamField>

<ParamField path="--validate" type="boolean">
  Validate LMDB after build
</ParamField>

## Global Flags

<ParamField path="--repo" type="string">

<ParamField path="--json" type="boolean" default="false">

## Common Workflows

### First-Time Setup

```bash
# Interactive setup with prompts
pairofcleats setup

# Non-interactive setup for CI
pairofcleats setup --non-interactive

# Fast bootstrap without prompts
pairofcleats bootstrap
```

### Building Indexes

```bash
# Build all modes
pairofcleats index build

# Build specific mode
pairofcleats index build --mode code

# Incremental rebuild
pairofcleats index build --incremental

# Watch for changes
pairofcleats index watch
```

### Searching Code

```bash
# Basic search
pairofcleats search "queryString"

# Search specific mode
pairofcleats search "queryString" --mode code

# Search with custom backend
pairofcleats search "queryString" --backend sqlite
```

## Exit Codes

| Code | Meaning |
|------|----------|
| `0` | Success |
| `1` | General error or validation failure |
| `2` | Invalid arguments |
| Signal | Terminated by signal (e.g., SIGTERM, SIGINT) |

## Environment Variables

PairOfCleats respects several environment variables:

- `PAIROFCLEATS_CACHE_ROOT` - Override cache directory
- `PAIROFCLEATS_CONFIG` - Path to configuration file
- `PAIROFCLEATS_TESTING` - Enable testing mode
- `NODE_ENV` - Node environment (development, production)

## Configuration

CLI behavior can be customized via `.pairofcleats.json` in your repository root. See the [Configuration Guide](/configuration/config-file) for details.

## Getting Help

```bash
# General help
pairofcleats --help

# Command-specific help
pairofcleats index --help
pairofcleats search --help
```
