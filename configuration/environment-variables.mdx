---
title: Environment Variables
description: Environment variables for configuring PairOfCleats behavior at runtime
---

PairOfCleats supports environment variables for runtime configuration, secrets, and testing. These override values from the configuration file.

## Secrets

<ParamField path="PAIROFCLEATS_API_TOKEN" type="string">
  Bearer token for API/MCP authentication when auth is enabled.
  
  ```bash
  export PAIROFCLEATS_API_TOKEN="your-secret-token"
  ```
</ParamField>

## Runtime Configuration

### Cache and Storage

<ParamField path="PAIROFCLEATS_HOME" type="string">
  Override the PairOfCleats home directory. Defaults to platform-specific locations:
  - Windows: `%LOCALAPPDATA%\PairOfCleats`
  - macOS/Linux: `~/.local/share/pairofcleats`
</ParamField>

<ParamField path="PAIROFCLEATS_CACHE_ROOT" type="string">
  Override the cache root directory. Takes precedence over config file setting.
  
  ```bash
  export PAIROFCLEATS_CACHE_ROOT="/custom/cache/path"
  ```
</ParamField>

<ParamField path="PAIROFCLEATS_CACHE_NAMESPACE" type="string">
  Cache namespace for isolation. Useful for parallel builds or testing.
</ParamField>

<ParamField path="PAIROFCLEATS_CACHE_REBUILD" type="string">
  Force cache rebuild when set to non-empty value.
</ParamField>

<ParamField path="PAIROFCLEATS_CACHE_METRICS_SAMPLE_RATE" type="number">
  Sampling rate for cache metrics (0.0 to 1.0).
</ParamField>

### MCP Configuration

<ParamField path="MCP_MODE" type="string">
  Alias for `PAIROFCLEATS_MCP_MODE`. Values: `legacy`, `sdk`, `auto`.
</ParamField>

<ParamField path="PAIROFCLEATS_MCP_MODE" type="string">
  MCP server mode. Requires @modelcontextprotocol/sdk capability for `sdk` mode.
  
  Values:
  - `legacy` - Legacy MCP protocol
  - `sdk` - Modern SDK-based protocol
  - `auto` - Automatic detection (default)
</ParamField>

<ParamField path="PAIROFCLEATS_MCP_TRANSPORT" type="string">
  MCP transport mechanism (stdio, http, etc.).
</ParamField>

<ParamField path="PAIROFCLEATS_MCP_QUEUE_MAX" type="number">
  Maximum queued MCP requests.
</ParamField>

<ParamField path="PAIROFCLEATS_MCP_MAX_BUFFER_BYTES" type="number">
  Maximum buffer size for MCP messages in bytes.
</ParamField>

<ParamField path="PAIROFCLEATS_MCP_TOOL_TIMEOUT_MS" type="number">
  Default timeout for MCP tool calls in milliseconds.
</ParamField>

### Runtime Tuning

<ParamField path="PAIROFCLEATS_THREADS" type="number">
  Number of worker threads for parallel operations.
  
  ```bash
  export PAIROFCLEATS_THREADS=8
  ```
</ParamField>

<ParamField path="PAIROFCLEATS_BUNDLE_THREADS" type="number">
  Number of threads for bundle operations.
</ParamField>

<ParamField path="PAIROFCLEATS_WORKER_POOL" type="string">
  Worker pool configuration.
</ParamField>

<ParamField path="PAIROFCLEATS_NODE_OPTIONS" type="string">
  Additional Node.js command-line options.
  
  ```bash
  export PAIROFCLEATS_NODE_OPTIONS="--max-old-space-size=8192"
  ```
</ParamField>

<ParamField path="PAIROFCLEATS_MAX_OLD_SPACE_MB" type="number">
  Node.js old-space heap size limit in megabytes.
</ParamField>

<ParamField path="PAIROFCLEATS_UV_THREADPOOL_SIZE" type="number">
  UV_THREADPOOL_SIZE override for libuv operations.
</ParamField>

<ParamField path="PAIROFCLEATS_IO_OVERSUBSCRIBE" type="boolean">
  Allow I/O operations to exceed CPU core count. Set to `1` or `true` to enable.
</ParamField>

<ParamField path="PAIROFCLEATS_LOG_FORMAT" type="string">

</ParamField>

<ParamField path="PAIROFCLEATS_STAGE" type="string">

## Test-Only Variables

<Warning>
These variables require `PAIROFCLEATS_TESTING=1` to be set and are only for development/testing.
</Warning>

<ParamField path="PAIROFCLEATS_TESTING" type="string">
  Enable test mode. Must be set to `1` for other test variables to take effect.
  
  ```bash
  export PAIROFCLEATS_TESTING=1
  ```
</ParamField>

<ParamField path="PAIROFCLEATS_TEST_CONFIG" type="string">
  JSON object string for test configuration override.
  
  ```bash

  ```
</ParamField>

<ParamField path="PAIROFCLEATS_TEST_MAX_JSON_BYTES" type="number">

<ParamField path="PAIROFCLEATS_TEST_ALLOW_MISSING_COMPAT_KEY" type="string">

<ParamField path="PAIROFCLEATS_TEST_STUB_DOCX_EXTRACT" type="string">

<ParamField path="PAIROFCLEATS_TEST_STUB_PDF_EXTRACT" type="string">

<ParamField path="PAIROFCLEATS_TEST_STUB_PDF_EXTRACT_DELAY_MS" type="number">

<ParamField path="PAIROFCLEATS_TEST_ALLOW_TIMEOUT_TARGET" type="string">

<ParamField path="PAIROFCLEATS_TEST_CACHE_SUFFIX" type="string">

<ParamField path="PAIROFCLEATS_TEST_CODE_MAP_BUDGET_MS" type="number">

<ParamField path="PAIROFCLEATS_TEST_NODE_OPTIONS" type="string">

## Precedence

Configuration precedence (highest to lowest):

1. CLI flags (e.g., `--threads 8`)
2. Environment variables (e.g., `PAIROFCLEATS_THREADS=8`)
3. Configuration file (`.pairofcleats.json`)
4. Defaults
